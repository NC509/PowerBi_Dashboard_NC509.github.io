## Objective
- In this project we want to show the coral bleaching levels that are objerved in different parts of the world.
- This dashboard needs to have a way for others to explore the changes of temperature and bleaching levels throughout time.
- Lastly, we want to understand what areas are at risk of coral bleaching based on average temperature.
